Welcome to Job Application Assistant

Applying for jobs often means rewriting the same details over and over. This extension helps you create tailored resume and cover letter drafts for specific job postings.

Upload an existing resume or cover letter, or build your profile from scratch. Your saved details guide the assistant when creating each draft.

Set up AI generation

To create real tailored drafts, Job Application Assistant needs an AI provider.

You can use local AI with Ollama, which runs on your own computer and does not require a paid API key.

Or you can connect a cloud AI provider using your own API key. Cloud providers are often faster and may give better results, but usage may cost money depending on your provider.

Demo Mode is still available if you only want to try the workflow, but it uses sample/mock generation rather than real AI output.
